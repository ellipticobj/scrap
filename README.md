# scrappad
this is a super simple vim-like notetaking app i made for myself

# features
super basic vim keybinds:
`:w` --- save
`:q` --- quit (`:q!` to force quit)
`:wq` -- write and quit
`i` ---- insert mode

default folder for notes is at ~/.config/scrappad
